The client needed a lead-generation website to promote their Design to WordPress Conversion services. It was important for the site to clearly communicate their value proposition, explain why clients should choose them, display transparent pricing and success stories, and provide visitors with a direct and simple way to get in touch.

### The theme:

- **Bricks Builder** - Bricks Builder is a modern, high-performance visual site builder for WordPress that empowers you to build beautiful, custom, and fast websites faster than coding from scratch, while giving you more control and cleaner code than traditional drag-and-drop builders.

### Plugins I used to create this website:

- **Automatic.css** - The #1 Utility Framework for WordPress Page Builders.
- **Advanced Custom Fields** - Customize WordPress with powerful, professional and intuitive fields.
- **LiteSpeed Cache** - High-performance page caching and site optimization from LiteSpeed
- **Local Google Fonts** - Host your used Google fonts on your server and make your site GDPR compliant.
- **Rank Math SEO** - Rank Math SEO is the Best WordPress SEO plugin with the features of many SEO and AI SEO tools in a single package to help multiply your SEO traffic.
- **Site Kit by Google** - Site Kit is a one-stop solution for WordPress users to use everything Google has to offer to make them successful on the web.
- **Code Snippets** - An easy, clean and simple way to run code snippets on your site. No need to edit to your theme's functions.php file again!
- **Add Featured Image Column** - This plugin adds a featured image column to the WordPress admin.
- **Admin and Site Enhancements (ASE)** -
  Easily enable enhancements and features that usually require multiple plugins.
- **Broken Link Checker by AIOSEO** - Monitor and test all internal and external links on your site for broken links. By AIOSEO, the original SEO plugin for WordPress.
- **Classic Editor** - Enables the WordPress classic editor and the old-style Edit Post screen with TinyMCE, Meta Boxes, etc. Supports the older plugins that extend this screen.
- **Image Alt Text** - Image Alt Text plugin provides facilities to add missing alt text for all image media files.
